## ExtraTime Design System

Use a light theme


Color Palette:
- Primary: #6366f1 (main actions, links, active states)
- Secondary: #8b5cf6 (secondary actions, accents)
- Accent: #f59e0b (highlights, special elements, pending states)
- Success: #10b981 (wins, correct predictions, positive feedback)
- Warning: #f59e0b (caution states, deadlines)
- Error: #ef4444 (losses, errors, wrong predictions)

Light Theme Colors:
- Background: #f8fafc
- Surface: #ffffff
- Text: #0f172a
- Muted: #64748b


### Design Principles
1. **Playful & Gamified**: Duolingo-inspired celebrations, streaks, achievements
2. **Sports-focused**: FotMob-style data presentation, match cards, scores
3. **Social & Engaging**: Discord-like community feel, fun interactions

### Component Guidelines
- Cards: 12px radius, medium shadow
- Buttons: Bold, clear CTAs with hover lift effect
- Badges: rounded style for status indicators
- Progress bars: Gradient fills for engagement
- Animations: Pulse, bounce, glow effects for celebrations

### Gamification Elements
- Streak counters with fire emoji and gradient backgrounds
- Achievement badges with category icons
- Level indicators with crown icons
- Progress bars with gradient fills
- Points display with large, bold numbers
- Celebration animations (confetti on wins)
